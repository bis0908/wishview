# WishView - 위시켓 프라이빗 매칭 프로젝트 뷰어

위시켓 프라이빗 매칭 프로젝트의 숨겨진 업무 내용을 확인할 수 있는 크롬 확장 프로그램입니다.

## 📋 프로젝트 개요

**WishView**는 위시켓의 프라이빗 매칭 기능으로 인해 제한된 프로젝트 정보를 JSON-LD 스키마를 통해 추출하여 사용자에게 표시하는 Chrome Extension입니다.

### 주요 기능

- ✅ **프라이빗 프로젝트 자동 감지**: `float-private-box` 클래스 탐지
- 🔍 **JSON-LD 데이터 추출**: JobPosting 스키마에서 프로젝트 정보 파싱
- 🎨 **직관적인 모달 UI**: 위시켓 디자인과 조화로운 인터페이스
- 💾 **프로젝트 저장 기능**: 관심 프로젝트 로컬 저장
- 📊 **사용 통계**: 조회 기록 및 활용 현황
- ⚙️ **사용자 설정**: 자동 표시, 알림 등 개인화 옵션

## 🏗️ 기술 스택

- **Runtime**: Chrome Extension Manifest V3
- **Language**: JavaScript (ES2020+)
- **Styling**: CSS3 (Grid, Flexbox, Custom Properties)
- **Architecture**: 모듈화된 컴포넌트 구조

## 🚀 설치 및 테스트

### 1. 개발자 모드에서 설치

1. Chrome 브라우저에서 `chrome://extensions/` 이동
2. 우측 상단의 **"개발자 모드"** 토글 활성화
3. **"압축해제된 확장 프로그램 로드"** 클릭
4. `wishview` 폴더 선택

### 2. 테스트 방법

#### 기본 동작 테스트

1. [위시켓 프로젝트 페이지](https://www.wishket.com/project/)로 이동
2. 프라이빗 매칭 프로젝트 선택 (🔒 아이콘이 있는 프로젝트)
3. 프로젝트 상세 페이지에서 자동으로 모달이 표시되는지 확인

#### 수동 실행 테스트

1. 위시켓 프로젝트 페이지에서 확장 아이콘 클릭
2. 팝업에서 **"프라이빗 정보 보기"** 버튼 클릭
3. 모달창이 표시되고 프로젝트 정보가 올바르게 표시되는지 확인

#### 기능별 테스트

- **탭 전환**: 모달 내 "업무 내용", "상세 정보", "회사 정보" 탭 동작
- **프로젝트 저장**: "저장하기" 버튼으로 프로젝트 저장 기능
- **설정**: 팝업에서 자동 표시, 알림 등 설정 변경
- **반응형**: 브라우저 창 크기 변경 시 모달 UI 적응

### 3. 디버깅

#### 콘솔 로그 확인

- **Content Script**: 페이지에서 F12 → Console 탭
- **Background Script**: `chrome://extensions/` → "검사" 클릭
- **Popup**: 팝업 열고 우클릭 → "검사"

#### 일반적인 문제 해결

1. **모달이 표시되지 않는 경우**
   - 위시켓 프로젝트 페이지인지 확인
   - `float-private-box` 클래스가 존재하는지 확인
   - Console에서 오류 메시지 확인

2. **JSON-LD 파싱 실패**
   - 페이지에 `application/ld+json` 스크립트 태그가 2개 이상 있는지 확인
   - 두 번째 스크립트가 JobPosting 타입인지 확인

3. **스타일 충돌**
   - 위시켓 페이지의 CSS와 충돌하는지 확인
   - 모달 z-index(999999) 우선순위 확인

## 📊 성능 고려사항

### 메모리 사용량

- Content Script: ~50KB
- Background Script: ~30KB
- 저장된 데이터: 사용자당 ~1MB 이하

### 실행 조건

- 위시켓 프로젝트 페이지에서만 동작
- DOM 로딩 완료 후 실행 (`document_idle`)
- 프라이빗 프로젝트 감지 시에만 활성화

## 🛡️ 보안 및 개인정보

### 데이터 수집

- **수집하지 않음**: 개인정보, 로그인 정보, 브라우징 기록
- **로컬 저장만**: 프로젝트 저장, 사용자 설정, 사용 통계
- **외부 전송 없음**: 모든 데이터는 사용자 기기에만 저장

### 권한 사용

```json
{
  "permissions": [
    "activeTab",    // 현재 탭 접근 (사용자 액션 시에만)
    "storage",      // 로컬 스토리지 (설정 및 저장 프로젝트)
    "scripting"     // 콘텐츠 스크립트 동적 삽입
  ],
  "host_permissions": [
    "https://www.wishket.com/*"  // 위시켓 도메인에서만 동작
  ]
}
```

## 🐛 알려진 이슈

1. **위시켓 페이지 구조 변경**: 사이트 업데이트 시 동작하지 않을 수 있음
2. **JSON-LD 형식 변경**: 위시켓의 데이터 구조 변경 시 파싱 실패 가능
3. **브라우저 호환성**: Chrome/Edge 외 브라우저에서는 테스트되지 않음
